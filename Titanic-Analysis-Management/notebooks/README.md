# 🧮 Notebooks

Titanic分析の実験・分析コードを記録するフォルダです。  
すべてのノートブックは「Input → Process → Output」の流れで整理されます。

---

## 📁 構成（推奨例）

| ファイル名 | 目的 | 説明 |
|-------------|------|------|
| `01_EDA.ipynb` | 探索的データ分析 | データ理解・欠損・相関・分布の把握 |
| `02_Preprocessing.ipynb` | 前処理 | 欠損補完・エンコード・正規化など |
| `03_Modeling.ipynb` | 学習 | モデル構築・評価・可視化 |
| `04_Ensemble.ipynb` | 改善 | アンサンブルや最適化の検証 |
| `05_Submission.ipynb` | 提出 | テストデータ予測と提出ファイル生成 |

---

## 📘 命名ルール

- ファイル名は `番号_処理内容.ipynb`
- 各ノートブックの冒頭に「目的・入力・出力・更新履歴」を記載
- 重い処理は `utils/` の関数に切り出す
