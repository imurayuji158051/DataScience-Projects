# 📊 Reports

本フォルダでは、Titanic分析における **結果・考察・可視化出力** を管理します。  
学習履歴・スコア・改善案などを体系的に記録し、再現性を高めることを目的としています。

---

## 📁 構成（推奨例）

| ファイル名 | 内容 |
|-------------|------|
| `score_log.csv` | モデル名・CVスコア・LBスコア・日付などの記録 |
| `feature_importance.png` | 特徴量重要度の可視化 |
| `learning_curve.png` | 学習曲線・過学習の確認 |
| `evaluation_report.xlsx` | 検証結果・指標・考察のまとめ |
| `summary.md` | 実験全体の要約・振り返り |

---

## 🧠 記録の観点（推奨）

- **精度（Accuracy / F1 / AUC）**
- **過学習傾向**
- **特徴量の影響度**
- **モデル比較**
- **改善提案と次の一手**

---

## 🧾 更新ルール

| 更新内容 | 記載場所 |
|-----------|-----------|
| 主要なスコア・改善記録 | `evaluation_report.xlsx` |
| 可視化出力 | `.png` ファイル群 |
| 全体のまとめ・考察 | `summary.md` |
